history object


					history.length
=> the number of pages in the history stack 

					history.back()
=> move back one page in the session history 

					history.forward()
=> move forward one page in the session history 

							history.go(2)
=> go two pages forward 

							history.go(-2)
=> go two pages backward 

							history.go(0)
